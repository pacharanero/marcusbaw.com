# marcusbaw.com

A personal portfolio website built with [Hugo](https://gohugo.io/) and [StaticCMS](https://www.staticcms.org)

## Development

1. Install `hugo`
2. Whilst inside the top directory, run:

```
hugo server
```

This will start a live-reload, local server.
